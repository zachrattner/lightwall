# Lightwall Personalities

This directory contains the various "personality" configurations for the Lightwall AI. These JSON files define the system prompts, response styles, and behavioral parameters that govern how the installation interacts with visitors.

## Active Personality

- `lightwall.json`: This is the primary configuration used in the installation. It has been tuned to provide a sensitive, perceptive, and expressive presence that aligns with the artistic vision of the piece.

## Other Contenders

The following personalities were developed during the design process. While not used in the intallation, they represent different experimental directions for the installation's character:

- `poetic.json`: Focused on lyrical and metaphoric responses.
- `robot.json`: A more literal, technical persona.
- `sage.json`: Provides wise, philosophical interactions.
- `sassy.json`: A more playful and opinionated character.
- `silly-robot.json`: A whimsical and less serious take on the robotic persona.
- `standard.json`: A baseline configuration for testing core functionality.
- `teacher.json`: Designed to be instructional and informative about the system.
